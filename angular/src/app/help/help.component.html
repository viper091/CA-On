Como podemos ajudar ?

O site é gratuito?

Sim, o nosso site é 100% gratuito.

 Como se cadastrar? Para se cadastras basta clicar na parte superior direita do site em “Registrar” e preencher os campos necessários. (Nome, E-mail, Senha, Confirmar Senha, Data de Nascimento, Estado e Cidade). Como ver minha carteirinha? Para visualizar a sua carteirinha após logado basta clicar em “minha carteirinha” no canto esquerdo. Se eu mudar de município eu vou continuar com meu cadastro? Sim, apenas se o sistema estiver disponível no seu município. Como recuperar minha senha em caso de esquecimento? Para trocar a senha basta clicar em “esqueci minha senha” colocar o seu e-mail e então será enviado um código no seu e-mail, então preencha o campo com o código e digite uma nova senha. Como fazer o cadastro para crianças? Para o cadastro de criança o responsável deve criar um e-mail para ela para fazer o seu cadastro. Não tem nada na minha carteira, o que fazer? Caso os campos de sua carteirinha esteja vazio, você deve ir ao postinho mais próximo para registrar/atualizar suas informações.
